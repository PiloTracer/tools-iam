# Directory Map

## Root
- `bin/`: Helper scripts (`start.sh`)
- `docker-compose*.yml`: Orchestration configs
- `.env*`: Environment configuration
- `.claude/`: Project context and AI agent memory

## .claude
- `agents/`: Agent personas
- `features/`: Feature specifications
- `plans/`: Implementation plans
- `decisions/`: ADRs

## Documentation
- `DOCS_CONTEXT.md`: High-level project context
- `DOCS_TECH_STACK.md`: Technology stack definitions
